# Каталог практических курсов

## 1. Python Backend с FastAPI
- Уровень: Intermediate
- Модули: Python Core, FastAPI, PostgreSQL, Auth, Deploy
- Задания: code
- Пример: реализовать POST /orders с валидацией и тестами

## 2. React + TypeScript для продукта
- Уровень: Intermediate
- Модули: TypeScript, React architecture, API integration, testing
- Задания: code + UI
- Пример: построить страницу каталога и покрыть smoke-тестами

## 3. DevOps: Docker + CI/CD + Monitoring
- Уровень: Advanced
- Модули: Docker, Compose, CI/CD, Observability
- Задания: deploy + pipeline

## 4. UI/UX Design Практика
- Уровень: Beginner
- Модули: UX research, wireframes, design system, handoff
- Задания: essay/case

## 5. Data Science: прикладной ML
- Уровень: Advanced
- Модули: EDA, feature engineering, models, metrics
- Задания: notebook/code/report

## 6. QA Engineer Pro
- Уровень: Intermediate
- Модули: test design, API/UI automation, CI integration
- Задания: тест-кейсы + автотесты
